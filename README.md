## 📚 项目介绍

> 此项目主要是为了帮助初学 Go 的同学，快速掌握面试常见内容，同时对于有经验的同学，也可以作为一个整理好的参考资料。

> 除此之外，我们还将提供一份完整的**Go 官方文档的翻译**，以方便大家查阅。
